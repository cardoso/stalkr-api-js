## Installation

```sh
# Install via yarn
yarn add stalkr-api-js-client

# or npm
npm install stalkr-api-js-client
```

## Usage

```js
// Use our default export:
import StalkrClient from 'stalkr-api-js-client';

// Or just the parts you need:
import {Users} from 'stalkr-api-js-client';
```
